    Create src/dispatch/callback-delivery.ts with:

    1. **Constants:**
       - `TERMINAL_STATUSES = new Set(["done", "cancelled", "deadletter"])`
       - `CALLBACK_TIMEOUT_MS = 120_000` (2 min per user decision)
       - `MAX_DELIVERY_ATTEMPTS = 3`
       - `MIN_RETRY_INTERVAL_MS = 30_000` (skip retries within 30s)

    2. **`DeliverCallbacksOptions` interface:**
       - taskId, store (ITaskStore), subscriptionStore (SubscriptionStore), executor (GatewayAdapter), logger (EventLogger), tracePath? (string)

    3. **`deliverCallbacks(opts)` function:**
       - Read task via store.get(taskId). Return early if not found or not terminal.
       - List active subscriptions via subscriptionStore.list(taskId, { status: "active" }).
       - Filter to granularity === "completion" (GRAN-01).
       - For each matching subscription, call deliverSingleCallback() wrapped in try/catch (DLVR-04).

    4. **`deliverSingleCallback(task, sub, opts)` private function:**
       - Build callback prompt via buildCallbackPrompt().
       - Build TaskContext for the callback session: agent = sub.subscriberId, taskId = original task ID, routing = { agent: sub.subscriberId }, priority = "normal".
       - Spawn session via executor.spawnSession(context, { timeoutMs: CALLBACK_TIMEOUT_MS, correlationId: randomUUID(), onRunComplete: async callback that calls captureTrace (DLVR-03) }).
       - If spawn succeeds: update subscription status to "delivered", set deliveredAt. Emit subscription.delivered event.
       - If spawn fails: increment deliveryAttempts, set lastAttemptAt. If deliveryAttempts >= MAX_DELIVERY_ATTEMPTS, set status "failed" + failureReason. Emit subscription.delivery_failed event.

    5. **`retryPendingDeliveries(opts)` function:**
       - Same options as deliverCallbacks.
       - List active subscriptions with deliveryAttempts > 0 (already attempted).
       - Filter to completion granularity.
       - Skip if lastAttemptAt is within MIN_RETRY_INTERVAL_MS of now.
       - For each eligible, call deliverSingleCallback() wrapped in try/catch.

    6. **`buildCallbackPrompt(task, sub, tracePath?)` exported function:**
       - Returns structured string per user decision: "You are receiving a task notification callback." prefix.
       - Includes taskId, title, finalStatus, subscriberId, trace path (if provided).
       - Extracts "## Outputs" section from task.body if present.

    7. **`extractOutputsSection(body)` helper:**
       - Find "## Outputs" marker in body. Extract text until next "## " marker or end of string.
