  <files>src/packaging/snapshot.ts, src/packaging/__tests__/snapshot.test.ts, src/schemas/config.ts, src/schemas/task.ts, src/schemas/org-chart.ts, src/schemas/project.ts</files>
  <behavior>
    - createSnapshot(aofRoot) copies entire data directory to `.aof/snapshots/snapshot-{timestamp}/`, excluding `.aof/snapshots/` itself to prevent recursive nesting
    - restoreSnapshot(aofRoot, snapshotPath) removes all non-snapshot content from aofRoot and copies snapshot contents back
    - pruneSnapshots(aofRoot, keep=2) removes oldest snapshots when more than `keep` exist
    - createSnapshot on empty dir creates a valid snapshot
    - createSnapshot on dir with .aof/snapshots/ excludes nested snapshots
    - restoreSnapshot replaces current data with snapshot data
    - pruneSnapshots keeps only the N most recent snapshots
    - schemaVersion z.union([z.literal(1), z.literal(2)]) passes validation for both values in config.ts, task.ts, org-chart.ts
    - ProjectManifest accepts optional defaultWorkflow string field
  </behavior>
  <action>
    1. Create `src/packaging/snapshot.ts` with three exported functions:
       - `createSnapshot(aofRoot: string): Promise<string>` -- Uses `node:fs/promises` `cp()` with `{recursive: true}`. Creates `.aof/snapshots/snapshot-{Date.now()}/`. Iterates top-level entries of aofRoot; for `.aof`, copies all subdirs EXCEPT `snapshots/`. Returns the snapshot path.
       - `restoreSnapshot(aofRoot: string, snapshotPath: string): Promise<void>` -- Removes all top-level entries in aofRoot EXCEPT `.aof/snapshots/`. Then copies everything from snapshotPath back to aofRoot, merging into `.aof/` without touching `snapshots/`.
       - `pruneSnapshots(aofRoot: string, keep: number = 2): Promise<void>` -- Reads `.aof/snapshots/`, sorts by name (timestamp-based), removes oldest entries until only `keep` remain.

    2. Create `src/packaging/__tests__/snapshot.test.ts` with tests for all three functions using temp directories.

    3. Relax schemaVersion in three schema files -- change `z.literal(1)` to `z.union([z.literal(1), z.literal(2)])`:
       - `src/schemas/config.ts` line 66
       - `src/schemas/task.ts` line 93
       - `src/schemas/org-chart.ts` line 313

    4. Add `defaultWorkflow` to `src/schemas/project.ts` ProjectManifest:
       ```typescript
       defaultWorkflow: z.string().optional(),
       ```
       Add it after the `workflowTemplates` field (line 138).
  </action>
