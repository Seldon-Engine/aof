<objective>
Make DAG workflows the default for new tasks in configured projects by auto-attaching the project's `defaultWorkflow` template during `bd create`, with `--no-workflow` opt-out and graceful degradation for unconfigured projects.

Output: Modified CLI that resolves default workflows automatically, with tests covering all precedence scenarios.
</objective>
