Create `scripts/verify-tarball.mjs` as a standalone Node.js script (not vitest). It complements `scripts/build-tarball.mjs` and is designed to run in CI between build and upload steps.

**Script structure:**

1. **Usage check**: Read `process.argv[2]` as the tarball path. If missing, print usage: `"Usage: node scripts/verify-tarball.mjs <tarball-path>"` and `process.exit(1)`.

2. **Constants**: `MAX_SIZE_MB = 15` (generous threshold -- safety net for accidentally shipping large binaries, not a precision measurement). This is Claude's discretion per CONTEXT.md.

3. **Step 1 -- Size check**: Use `statSync(tarball)` to get file size. If `size > MAX_SIZE_MB * 1024 * 1024`, print `"FAIL: Tarball size ${sizeMB}MB exceeds ${MAX_SIZE_MB}MB threshold"` and `process.exit(1)`. Print `"PASS: Size ${sizeMB}MB (max ${MAX_SIZE_MB}MB)"`.

4. **Step 2 -- Extract**: Create temp dir with `mkdtempSync(join(tmpdir(), "aof-verify-"))`. Run `execSync("tar -xzf " + tarball + " -C " + extractDir)`. If extraction fails, print `"FAIL: Extraction failed"` and `process.exit(1)`. Print `"PASS: Extraction"`.

5. **Step 3 -- Required files**: Check that `package.json`, `dist/cli/index.js`, `openclaw.plugin.json` exist in the extracted directory using `existsSync`. If any missing, print `"FAIL: Missing required file: ${file}"` and `process.exit(1)`. Print `"PASS: Required files present"`.

6. **Step 4 -- npm ci --production**: Run `execSync("npm ci --production --ignore-scripts", { cwd: extractDir, stdio: "pipe" })`. If fails, print `"FAIL: npm ci --production failed"` and `process.exit(1)`. Print `"PASS: npm ci --production"`.

7. **Step 5 -- CLI boot**: Run `execSync("node dist/cli/index.js --version", { cwd: extractDir, encoding: "utf-8" })`. Capture stdout. If fails, print `"FAIL: CLI failed to boot"` and `process.exit(1)`. Print `"PASS: CLI boots"`.

8. **Step 6 -- Version match**: Read `package.json` from extracted dir, extract `version`. Compare with the CLI `--version` output (trim both). If they don't match, print `"FAIL: Version mismatch: package.json=${pkgVersion}, CLI=${cliVersion}"` and `process.exit(1)`. Print `"PASS: Version match (${pkgVersion})"`.

9. **Cleanup**: Remove temp dir with `rmSync(extractDir, { recursive: true, force: true })`.

10. **Summary**: Print `"\nAll checks passed."` and exit with code 0.

**Output format**: Each step prints `"PASS: ..."` or `"FAIL: ..."` on its own line. Consistent with the smoke command's checklist format. Use plain text (no ANSI colors) since this runs in CI logs.

**Imports**: Use `execSync` from `node:child_process`, `statSync, existsSync, mkdtempSync, readFileSync, rmSync` from `node:fs`, `join` from `node:path`, `tmpdir` from `node:os`.

**Error handling**: Wrap each step in try/catch. On any failure, clean up the temp dir before exiting. Use a cleanup function or try/finally to ensure temp dir is removed.
