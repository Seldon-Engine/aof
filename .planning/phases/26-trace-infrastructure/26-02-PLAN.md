  <action>
    **1. Add event types to `src/schemas/event.ts`:**

    - `"trace.captured"` -- emitted after successful trace write
    - `"trace.capture_failed"` -- emitted when trace capture fails

    Place them after the existing `"completion.enforcement"` entry, grouped with a comment `// Trace lifecycle (Phase 26)`.

    **2. Create trace writer (`src/trace/trace-writer.ts`):**

    Export:
    ```typescript
    interface CaptureTraceOptions {
      taskId: string;
      sessionId: string;
      agentId: string;
      durationMs: number;
      store: ITaskStore;
      logger: EventLogger;
      debug: boolean;
    }

    interface CaptureTraceResult {
      success: boolean;
      noopDetected: boolean;
      tracePath?: string;
      error?: string;
    }

    async function captureTrace(opts: CaptureTraceOptions): Promise<CaptureTraceResult>
    ```

    Implementation flow:
    1. **Resolve session file path:** Use the known pattern `~/.openclaw/agents/<agentId>/sessions/<sessionId>.jsonl`. Normalize agentId by stripping `agent:` prefix if present (same normalization as OpenClawAdapter).
    2. **Parse session:** Call `parseSession(sessionFilePath, { debug: opts.debug })` from Plan 01's module.
    3. **Detect no-op:** Call `detectNoop({ toolCallCount: parsed.toolCallCount, sessionMissing: parsed.sessionMissing })`.
    4. **Determine attempt number:** Read `state/runs/<taskId>/` directory, filter for `trace-*.json` files, count them, next = count + 1. If directory doesn't exist, attempt = 1.
    5. **Build trace object:** Construct the full trace matching `TraceSchema`:
       - `version: 1`, `taskId`, `sessionId`, `attemptNumber`, `capturedAt: new Date().toISOString()`
       - `session: { sessionFilePath, durationMs, model, provider, thinkingLevel }`
       - `toolCalls`, `toolCallCount`, `reasoning` (from parser)
       - `noopDetected` from detector
       - `meta: { mode, unknownEntries, parseErrors, truncated, totalEntriesParsed }`
    6. **1MB cap (debug mode only):** Serialize to JSON, check `Buffer.byteLength`. If > 1_000_000 bytes, truncate tool call outputs to empty strings starting from the end, re-serialize, set `meta.truncated = true`. Repeat until under cap or all outputs truncated.
    7. **Write trace file:** `await mkdir(taskDir, { recursive: true })`, then `write-file-atomic` to `trace-{N}.json`.
    8. **Emit events:** `trace.captured` on success with `{ attemptNumber, toolCallCount, mode, tracePath }`. If noopDetected, also emit `completion.noop_detected` with `{ sessionId, toolCallCount: 0 }`.
    9. **Return result.**

    The entire function body is wrapped in try/catch. On any error, try to emit `trace.capture_failed` (also in try/catch -- even logging is best-effort), then return `{ success: false, noopDetected: false, error: message }`.

    **3. Write tests using mocked filesystem and logger:**

    Use `vi.mock` for `node:fs/promises` and `write-file-atomic`. Mock `parseSession` from `../session-parser.js` and `detectNoop` from `../noop-detector.js` to control their return values. Mock `EventLogger` to capture emitted events.

    Test cases per behavior list above.
