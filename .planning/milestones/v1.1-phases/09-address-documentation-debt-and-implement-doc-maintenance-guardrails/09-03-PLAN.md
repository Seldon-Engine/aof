- Type: OpenClaw plugin (not standalone)
- Install: `curl -fsSL https://raw.githubusercontent.com/d0labs/aof/main/scripts/install.sh | sh`

**Core value:** Tasks never get dropped — they survive gateway restarts, API failures, rate limits, and agent crashes, always resuming and completing end-to-end without human intervention.

**Key config files:**
- `org-chart.yaml` — defines agents, teams, skills, routing rules (schema in src/schemas/org-chart.ts, has 27 JSDoc blocks)
- AOF config — in OpenClaw's config, key fields: dataDir, polling interval, adapter settings (schema in src/schemas/config.ts)
- `openclaw.plugin.json` — plugin manifest with configSchema JSON Schema
- OpenClaw wiring — plugin registration in openclaw.json gateway config

**Architecture overview sources:**
- src/dispatch/ — scheduler, executor, assign-executor (dispatch pipeline)
- src/store/ — FilesystemTaskStore (task persistence)
- src/events/ — EventLogger, NotificationService (event sourcing)
- src/memory/ — VectorStore, HnswIndex (semantic memory)
- src/cli/ — Commander-based CLI
- src/daemon/ — AOFService (poll loop, lifecycle)
- src/openclaw/ — OpenClawAdapter (gateway integration)
- src/schemas/ — Zod schemas for tasks, org-chart, config, events, protocols
